# Events Log

Lightweight log for sickness, significant lifestyle disruptions, and other non-training events
that help explain anomalies in the activity cache and readiness data.

Format: `DATE  TYPE  Notes`

Types: SICK, CLEARED, TRAVEL, STRESS, ALCOHOL, POOR_SLEEP

Examples:
```
2025-03-06  SICK      Upper respiratory infection, sore throat, no fever
2025-03-14  CLEARED   Feeling normal, back to training
2025-06-01  TRAVEL    Flying to race, arrive late, poor sleep expected
```
